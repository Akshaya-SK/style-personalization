## V0 — Basic Questionnaire System
### Description:

A foundational system where users input basic attributes through a static questionnaire, and outputs are generated using simple rule-based logic.

#### Key Features:
- Static questionnaire (face, body, color indicators)
- Hardcoded rule-based classification
- Basic output:
- Face shape
- Body type
- Color category
### Objective:

- Validate input → processing → output pipeline
- Establish core backend + frontend integration
